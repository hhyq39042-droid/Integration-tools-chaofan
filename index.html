如何更新工具

自动更新通知机制
✓

工具中心每隔 3分钟（可在设置中调整）自动检查所有工具是否有新版本

✓

检测到更新时，对应 标签会出现红点，当前工具顶部会出现 黄色横幅 提示

✓

成员点击 「立即更新」 即可加载最新版本，无需管理员通知

管理员：更新工具文件（GitHub）
1

登录 github.com → 进入仓库 Integration-tools-chaofan

2

点击要更新的工具文件名 → 右上角点击 上传图标 ↑ → 上传新版本文件

3

底部点击 Commit changes 保存，等约 30秒~1分钟 生效

4

成员打开工具中心后，系统自动检测并提示更新，无需手动通知

添加新工具
1

上传新工具 HTML 到仓库（Add file → Upload files）

2

编辑 index.html 顶部 TOOLS 数组，新增一行：
{ id:5, name:'工具名', file:'文件名.html', emoji:'🔧' },

3

Commit 保存，约1分钟后新工具自动出现在工具中心
